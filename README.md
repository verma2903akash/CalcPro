# 🧮 CalcPro
A modern calculator application built using React Native and Expo SDK 54.
## Features
✅ Basic Calculator
✅ Scientific Calculator
✅ Unit Converter
✅ Currency Converter (Live Exchange Rates)
✅ Calculation History
✅ Notes Section
✅ Dark Theme UI
✅ Safe Area Support

## Tech Stack

- React Native
- Expo SDK 54
- AsyncStorage
- React Navigation
- JavaScript
